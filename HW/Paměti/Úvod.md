## Text Elements
Pameti - Uvod ^MpSzb8Cl

- Slouzi k uchovani a opetovanemu vyvolani informace
    - Binarni forma

- Polovodicove obvody/moduly
    - Ruzne vlastnosti dle typu konstrukce pametovych bunek

- Slozeny z pametovych bunek
    - 1 bit = 1 pametova bunka
 
- Maticove usporadani
    - Pametova mriza/matice
    - Poloha umisteni bunky

- Adresa
    - Souradnice radku a sloupce
    - Zadavana binarne na adresni sbernici
        - V textu vetsinou uvedena v hex

- Data
    - Datova sbernice
        - Jednosmerna nebo oboustranna

- Pametove misto
    - Nejmensi adresovatelna jednotka
    - Dana adresou a poctem pametovych bunek, ktere jsou zde ulozeny ^YjC3wu6P

Kapacita pameti ^QD7uEw4E

- Celkovy objem dat/informaci, ktere mohou byt v pameti ulozeny

- Pocet pametovych bunek, ktere pamet obsahuj

- Soucin hloubky pameti a delky datoveho slova

- Jednotka bit, resp. byte a jeho nasobky ^OyFjU2B2

Organizace pameti ^SAySuQZn

- Informace je usporadana do N pametovych mist, ktere se obvykle oznacuji
  jako slova s n bity
        - Kapacita pameti = N * n

- Jedno z N slov je vybrano pomoci p-bitove adresy
    - N = 2^p

- Pocet adresovych vodicu (p-bit adreasa)
    - p = logN/log2

- Priklad 1
    -14 adresovych vodicu, kolik je hloubka pameti
    - N = 2^p = 16384 = 16k
- Priklad 2
    - hloubka pameti = 128k slov, kolik potrebujeme vodicu
    - N = 2^p
    - p = log2(128k) = 17 ^5ykBhVGh
